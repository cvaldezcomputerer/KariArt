My Works

Help a teacher to start her online teaching buisness with a website.

YoCo English will provide English lessons to support immigrants from Spanish speaking countries starting a new life .

Target Audience

Through an interview with the teacher, I learned the frustrations the
teacher’s demographic struggle with when using websites.

  • Finding links to different pages on websites and finding how to log in.

  • Navigating through too much information.

  • Using Zoom links.

  • No explanations for why they need accounts.

The website’s main goals

  • When users visit the website they will be able to see information
    about the teaching service.

  • Zoom meetings can be accessed directly on the website.

  • Using Zoom links.

  • Lesson appointments can be booked and tracked.

Ideation

To keep a clear focus, a site map was created to help visualize the key
points of the website.

Items were redesigned until the final product.

Finding links to different pages on websites
and finding how to log in:

Links will be clearly stated with rounded buttons that contrast against
the elements. A navigation bar that scrolls with the user is provided.
The user can navigate to the desired page at any point.

Having to open new pages and unable to navigate: The nav bar will
scroll with the user. This will let the user press the desired page at
any moment. The user can access everything needed directly on the
web page.

Zoom links must be given directly on the website: The profile
option will store user’s Zoom sessions.

No explanations for why they need accounts: Pop up boxes explaining
certain features are provided.

What can be done differently?

  • The contrasting colors can be too harsh for the target audience.

  • I wasn’t able to test the website on the target audience to fix what
    causes frustrations.
