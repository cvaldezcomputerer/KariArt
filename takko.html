Takko Taco

UX/ UI / Prototyping

Overview

I created the interface for a custom-built website designed to help English learners in Takko, Aomori, Japan. The site offers resources like fun learning games, news updates, and information about Takko Town, all aimed at making English practice more accessible and enjoyable.
